# Plan My Day

Runs the **daily-brief** skill for today.

## Instructions

1. Read `$ARGUMENTS`. If it looks like a date, brief that day; if it looks like a theme, use it as the day's suggested focus. Otherwise default to today.
2. Run the **daily-brief** skill: pull today's ~~calendar, top priorities and overdue items from ~~project tracker, and unread items needing a reply from ~~email and ~~chat.
3. If no tools are connected, ask the user to paste their agenda and task list, then build the brief from that.
4. Return the one-screen brief and name a single suggested focus with a free block to protect.
