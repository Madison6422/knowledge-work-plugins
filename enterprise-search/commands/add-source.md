# Add Source

Runs the **source-management** skill: review coverage and add a source.

## Instructions
1. If the argument names a source, focus on it — what it adds and how to connect it. Otherwise review all sources.
2. Invoke the **source-management** skill.
3. Report which sources are connected, what each is best at finding, and where coverage gaps are.
4. Recommend the single highest-value source to connect next, and note that servers are pinned in `.mcp.json` (see `CONNECTORS.md`).
5. Point the user to **unified-search** to try a query once a new source is connected.
