# Escalate

Runs the **escalation-packet** skill to package an issue for engineering or product.

## Instructions
1. Take the input from `$ARGUMENTS` — a ~~helpdesk ticket ID to pull, or a pasted issue description.
2. Run the **escalation-packet** skill: build repro steps, quantify impact and affected customers, and collect links into a crisp summary.
3. Show the drafted ~~project tracker item and ~~chat notification for approval.
4. Do not file or post until a human confirms. On approval, file into the ~~project tracker and notify the right ~~chat channel.
